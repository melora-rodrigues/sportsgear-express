# sportsgear-express
A web-based platform for renting and managing sports equipment.
SportsGear Express is a web-based application for managing sports equipment rentals.Users can browse and view products, place rental orders, make online payments, and provide feedback. The platform also includes an admin interface for managing products, users, orders, and customer feedback.

This project was developed using HTML, CSS, and JavaScript for the frontend, PHP for backend development, and MySQL for the database. XAMPP was used as the local server environment, and phpMyAdmin was used for database management.
